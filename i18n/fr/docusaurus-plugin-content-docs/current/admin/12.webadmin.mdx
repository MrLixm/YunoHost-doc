---
title: ⭐ Interface d'admin web
---

YunoHost est fourni avec une interface graphique d’administration (aussi appelée webadmin). L’autre méthode d'administration est d’utiliser la [ligne de commande](/admin/command_line).

La webadmin est accessible à l’adresse `https://votredomaine.tld/yunohost/admin` ou via l'adresse IP locale ou globale (`https://1.2.3.4/yunohost/admin`)
